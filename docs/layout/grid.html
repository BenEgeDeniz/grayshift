Grid System

Build responsive layouts using Grayshift's powerful mobile-first flexbox grid.

How It Works

Grayshift's grid system uses a series of containers, rows, and columns to layout and align content:

  • Containers center and horizontally pad your content
  • Rows are wrappers for columns
  • Columns are where your content is placed
  • The grid is based on 12 columns

Equal Width Columns

Create equal-width columns with .col:

.col

Responsive Columns

Use responsive column classes with breakpoint prefixes:

.md:col-6
.md:col-6
.lg:col-4
.lg:col-4
.lg:col-4

  <div class="md:col-6">Half width on medium+</div>
  <div class="md:col-6">Half width on medium+</div>
</div>
<div class="row">
  <div class="lg:col-4">Third on large+</div>
  <div class="lg:col-4">Third on large+</div>
  <div class="lg:col-4">Third on large+</div>
</div>

Breakpoints

Grayshift uses these responsive breakpoints:

  • sm: - Small devices (≥576px)
  • md: - Medium devices (≥768px)
  • lg: - Large devices (≥992px)
  • xl: - Extra large devices (≥1200px)

Mixed Widths

Combine different column widths:

.md:col-8
.md:col-4
<div class="row">
  <div class="md:col-8">Wider column</div>
  <div class="md:col-4">Narrower column</div>
</div>
