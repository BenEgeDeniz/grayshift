Containers

Containers are the most basic layout element in Grayshift and are required when using the grid system.

Default Container

Use .container for a responsive fixed-width container:

.container 
<div class="container">
  <!-- Content here -->
</div>

The container width changes at each breakpoint:

  • Extra small (<576px): 100%
  • Small (≥576px): 540px
  • Medium (≥768px): 720px
  • Large (≥992px): 960px
  • Extra large (≥1200px): 1140px

Fluid Container

Use .container-fluid for a full-width container:

.container-fluid 
<div class="container-fluid">
  <!-- Content here -->
</div>

Spans the entire width of the viewport.

Nesting Containers

Containers can be nested, though most layouts don't require this:

<div class="container">
  <div class="row">
    <div class="col">
      <div class="container">
        <!-- Nested content -->
      </div>
    </div>
  </div>
</div>
