Modal

Use modals to add dialogs for lightboxes, user notifications, or custom content.

Basic Modal

Toggle a modal with data-gs-toggle="modal":

Modal title

Modal body text goes here.

<button data-gs-toggle="modal" data-gs-target="#exampleModal">
  Launch Modal
</button>

<div class="modal" id="exampleModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h3>Modal title</h3>
        <button class="btn-close" data-gs-dismiss="modal"></button>
      </div>
      <div class="modal-body">Modal body text</div>
      <div class="modal-footer">
        <button data-gs-dismiss="modal">Close</button>
      </div>
    </div>
  </div>
</div>

Modal Sizes

Modals have optional sizes with .modal-sm, .modal-lg, and .modal-xl:

<div class="modal-dialog modal-lg">...</div>
