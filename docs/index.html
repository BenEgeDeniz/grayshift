Why Choose Grayshift?

Modern, flexible, and easy to use component library

Lightweight

Minimal footprint with only the essentials. Fast loading times and optimal performance for modern web applications.

Customizable

Built with SCSS variables and mixins. Easily customize colors, spacing, and components to match your brand.

Responsive

Mobile-first design with responsive utilities. Build interfaces that work perfectly on any device or screen size.

Quick Start

Get started with Grayshift in seconds

1. Download or Clone

Download the latest release or clone the repository:

git clone https://github.com/yanchokraev/grayshift.git

2. Include CSS and JS

Add the following to your HTML:

<link rel="stylesheet" href="path/to/grayshift.min.css">
<script src="path/to/grayshift.min.js"></script>

3. Start Building

Use Grayshift components in your HTML:

<button class="btn btn-primary">Click Me</button>
