Installation

Learn how to include Grayshift in your project using various methods.

Download

Download the latest compiled and minified CSS and JavaScript files:

Download Grayshift v1.0.2

The download includes the following files in the dist/ folder:

  • css/grayshift.css - Uncompressed CSS
  • css/grayshift.min.css - Minified CSS (recommended)
  • js/grayshift.js - Uncompressed JavaScript
  • js/grayshift.min.js - Minified JavaScript (recommended)

Clone from GitHub

Clone the repository to get the complete source code including SCSS files:

git clone https://github.com/yanchokraev/grayshift.git

This method gives you access to:

  • Source SCSS files for customization
  • Source JavaScript files
  • Build tools and configuration
  • Examples and documentation

Include in Your HTML

Once you have the files, include them in your HTML document:

CSS

Add the CSS file in the <head> section:

<link rel="stylesheet" href="path/to/grayshift.min.css">

JavaScript

Add the JavaScript file before the closing </body> tag:

<script src="path/to/grayshift.min.js"></script>

Complete Example

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>My Grayshift Project</title>
  <link rel="stylesheet" href="path/to/grayshift.min.css">
</head>
<body>
  <div class="container">
    <h1>Hello, Grayshift!</h1>
    <button class="btn btn-primary">Click Me</button>
  </div>
  <script src="path/to/grayshift.min.js"></script>
</body>
</html>

Fonts (Optional)

Grayshift works with any font, but the examples use Inter. You can include it from Google Fonts:

<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Inter:wght@100;200;300;400;500;600;700;800;900&display=swap">

Source Files

If you want to customize Grayshift, you can work with the source SCSS files:

  1. Clone the repository or download the source files
  2. Install a SCSS compiler (like Sass or node-sass)
  3. Modify the SCSS variables in scss/_variables.scss
  4. Compile the SCSS to CSS

See the Quick Start guide for more details on customization.
