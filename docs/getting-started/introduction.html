Introduction

Get started with Grayshift, a lightweight front-end component library for developing fast and powerful web interfaces.

What is Grayshift?

Grayshift is a modern, lightweight front-end component library designed to help developers build beautiful and responsive web interfaces quickly. It provides a comprehensive collection of UI components, a flexible grid system, and utility classes that work seamlessly together.

Built with SCSS and vanilla JavaScript, Grayshift offers excellent performance with minimal overhead while maintaining maximum flexibility and customization options.

Key Features

  • Lightweight: Minimal file size with no unnecessary bloat
  • Responsive: Mobile-first design with responsive breakpoints
  • Customizable: Easy to customize with SCSS variables
  • Modern: Built with modern CSS and JavaScript
  • Accessible: ARIA-compliant components
  • Browser Support: Works in all modern browsers

Browser Support

Grayshift supports the latest, stable releases of all major browsers:

  • Chrome (latest)
  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
  • Opera (latest)
