Quick Start

Get up and running with Grayshift in just a few minutes.

Basic Template

Start with this basic HTML template that includes all necessary Grayshift files:

<!doctype html>
<html lang="en">
<head>
  <!-- Required meta tags -->
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- Grayshift CSS -->
  <link rel="stylesheet" href="path/to/grayshift.min.css">

  <!-- Optional: Google Fonts -->
  <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Inter:wght@100;200;300;400;500;600;700;800;900&display=swap">

  <title>Hello, Grayshift!</title>
</head>
<body>
  <div class="container py-5">
    <h1>Hello, Grayshift!</h1>
    <p>Get started with Grayshift.</p>
  </div>

  <!-- Grayshift JavaScript -->
  <script src="path/to/grayshift.min.js"></script>
</body>
</html>

Customization

To customize Grayshift, you can:

  1. Modify SCSS Variables: Edit scss/_variables.scss to change colors, spacing, and more
  2. Override Styles: Add your own CSS after the Grayshift stylesheet
  3. Use Utility Classes: Apply spacing, colors, and other utilities directly in HTML

Example: Custom Colors

// In your custom SCSS file
$blue: #1e40af;
$primary: var(--blue);

@import "path/to/grayshift.scss";
