Helpers & Utilities

Utility classes for quick styling and layout adjustments.

Spacing

Use margin and padding utilities to control spacing:

.m-*   - Margin on all sides
.mt-*  - Margin top
.mb-*  - Margin bottom
.ms-*  - Margin start (left)
.me-*  - Margin end (right)
.mx-*  - Margin left and right
.my-*  - Margin top and bottom

.p-*   - Padding on all sides
.pt-*  - Padding top
.pb-*  - Padding bottom
.ps-*  - Padding start (left)
.pe-*  - Padding end (right)
.px-*  - Padding left and right
.py-*  - Padding top and bottom

Sizes: 0, 1, 2, 3, 4, 5, 6

Example:

<div class="mb-3">Margin bottom 3</div>
<div class="p-4">Padding 4</div>

Display

Control element display:

.d-none         - Display none
.d-block        - Display block
.d-inline       - Display inline
.d-inline-block - Display inline-block
.d-flex         - Display flex
.d-inline-flex  - Display inline-flex

Flexbox

Flexbox utilities for alignment and justification:

.flex-row          - Flex direction row
.flex-column       - Flex direction column
.justify-content-start    - Justify start
.justify-content-center   - Justify center
.justify-content-end      - Justify end
.justify-content-between  - Justify space-between
.align-items-start        - Align start
.align-items-center       - Align center
.align-items-end          - Align end

Text

Text utilities:

.text-start    - Text align left
.text-center   - Text align center
.text-end      - Text align right
.text-white    - White text color
.fs-1 to .fs-6 - Font sizes

Sizing

Width and height utilities:

.w-25   - Width 25%
.w-50   - Width 50%
.w-75   - Width 75%
.w-100  - Width 100%

.h-25   - Height 25%
.h-50   - Height 50%
.h-75   - Height 75%
.h-100  - Height 100%

Other Helpers

Additional helpful utilities:

  • .visually-hidden - Visually hide an element
  • .list-unstyled - Remove list styling
  • .clearfix - Clear floats
  • .text-truncate - Truncate text with ellipsis
  • .rounded - Rounded corners
  • .rounded-circle - Circular element
